PRUEBA 1 – CRUD + Hooks (la más común)

Tiempo estimado: 1–1.5 h
Nivel: Junior–Mid

Enunciado

Construye una app en React que permita listar, crear, editar y eliminar usuarios.

Requisitos

Usar React Hooks (useState, useEffect)

Consumir una API pública (ej: https://jsonplaceholder.typicode.com/users)

Mostrar lista de usuarios

Formulario para:

Crear usuario (mock)

Editar usuario

Eliminar usuario de la lista (frontend)

Manejar loading y error

Bonus (si da tiempo)

Validaciones básicas

Componentes reutilizables

Separar lógica en hooks

Qué evalúan

Manejo de estado

Limpieza del código

Organización de componentes

Uso correcto de useEffect
